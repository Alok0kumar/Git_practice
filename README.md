# GITT

A project to manage and track your Git repositories efficiently.

## Features

- Easy repository management
- Visual commit history
- Branch and merge tracking

## Installation

```bash
git clone https://github.com/yourusername/gitt.git
cd gitt
```

## Usage

```bash
python main.py
```
